# MarketPlace-Pixel

[➡️ **Live Demo**](https://jithin-joy-arakkal.github.io/MarketPlace-Pixel/)

Pixel's Market Place — An immersive 3D virtual marketplace experience built entirely in the browser. Explore, interact, and shop using A-Frame and modern web technologies.

## Features
- **3D Marketplace Environment**: Navigate a pixel-art-inspired virtual world.
- **Interactive UI and Panels**: Intuitive overlays for location, inventory, and info.
- **Walkthrough with A-Frame**: Move around, interact with products and stalls in first-person view.
- **Inventory System**: Collect and view items in your personal inventory.
- **Mobile Friendly**: Fully responsive UI for all device types.
- **Customizable**: Edit products, shop entities, or world layout via HTML and JS.

## How It Works
- Use WASD/arrow keys and mouse/touch to navigate marketplace.
- Click/interact with items/booths; see your changes in inventory panel.
- Play instantly — no backend or downloads required.

## Getting Started
1. Clone the repository:
   ```sh
   git clone https://github.com/jithin-joy-arakkal/MarketPlace-Pixel.git
   ```
2. Open `index.html` in your browser, or visit the [Live Demo](https://jithin-joy-arakkal.github.io/MarketPlace-Pixel/).
